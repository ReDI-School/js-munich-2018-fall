What is a FOR loop?

The for loop is the kind of loop used the most.

Example:


                    for (let i = 0; i < 3; i++) {
                        // shows 0, then 1, then 2
                        alert(i);
                    }
                

                    for (begin; condition; step) {
                        // loop body
                    }

Initialization


                    for (let i = 0; i < 3; i++) {
                        // shows 0, then 1, then 2
                        alert(i);
                    }
  • In the example. The start condition is i = 0.
  • Traditionally we use i letter symbol for the variable.
  • Why 0 ?

Condition


                    for (let i = 0; i < 3; i++) {
                        // shows 0, then 1, then 2
                        alert(i);
                    }
  • Aka "Are we still going?"
  • Fancy way to say how long to loop for
  • A condition: true or false

Step


                    for (let i = 0; i < 3; i++) {
                        // shows 0, then 1, then 2
                        alert(i);
                    }
  • We need a false to break the loop.
  • Final reason to tell us how to go from starting point to terminating point.
  • ++ means to add 1 each time the loop runs

    • Another kind of loop

    The "while" loop

    While the condition is true, the code from the loop body is repeated.

    The syntax is:

    
                    while (condition) {
                        // code
                        // so-called "loop body"
                    }

    Let's try it!

    
                    let i = 0;
                    while (i < 3) { // shows 0, then 1, then 2
                        alert(i);
                        i++;
                    }

    A more common example

    
                    function game(player1, player2) {
                        let isRunning = true;

                        while(isRunning) {
                            updateGame();
                            drawGame();
                            if (didSomeoneWin()) {
                                isRunning = false;
                            }
                        }
                    }
