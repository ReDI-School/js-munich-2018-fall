getElementById

Will convert an HTML with the same id to a Javascript entity

Like every Javascript entity you can read/update/change its value

IMPORTANT: the id is Case Sentive

The id must be unique

If there are multiple elements with the same id, then the behavior of corresponding methods is unpredictable.

The browser may return any of them at random. So please stick to the rule and keep id unique.
