Lesson 7

Another kind of loop:

The "while" loop

While the condition is true, the code from the loop body is repeated.


    let keepGoing = "yes";
    while (keepGoing === "yes") { 
        keepGoing = prompt("Do you want to continue?");
    }

Lesson 8

Arrays

Arrays let you store multiple values in a single variable


    let fruit = ["Banana", "Apple", "Kiwi", "Lemon"];

Array access

You can access a particular element by passing its index to the array name within square brackets:


                        let films = ["Jurassic Park", "Titanic", "Toy Story"];

                        console.log(films[0]); // will be "Jurassic Park"
                        console.log(films[1]); // will be "Titanic"
                        console.log(films[2]); // will be "Toy Story"

Array & for loop

Arrays are great friends with for loops!

For loop lets you execute something for every element of an array.


    let fruit = ["Banana", "Apple", "Kiwi", "Lemon"];

    for (let i = 0; i < fruit.length; i++) {
        alert("I like " + fruit[i]);
    }

Add to an array

Use push to add new element to an array


    let fruit = [];

    let favoriteFruit = prompt("What kind of fruit do you like?");
    while(favoriteFruit !== "") {
        fruit.push(favoriteFruit);
        favoriteFruit = prompt("What kind of fruit do you like?");
    }

    for (let i = 0; i < fruit.length; i++) {
        console.log("You like " + fruit[i]);
    }

Pure functions

  • Depend only on arguments
  • No side effects

                    let x = 5, y = 10;
    
                    function sum() {
                        alert(x + y);
                    }

vs


                    function sum(x, y) {
                        return x + y;
