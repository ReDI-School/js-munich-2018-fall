How do you run a function when a button is clicked?

                    let buttonEl = ...;
                    function onClick() { ... };
                    buttonEl.addEventListener("click", onClick);
How do you run a function when an input is changed?

                    let inputEl = ...;
                    function onInput() { ... };
                    inputEl.addEventListener("input", onInput);

How do you run a function when a key is pressed inside an input?

                    let inputEl = ...;
                    function onKeyPress() { ... };
                    inputEl.addEventListener("keypress", onKeyPress);

How can you get more information about the event in an event listener?

With the event object! Simply receive it as an argument in your listener: 

                        function onKeyPress(event) {
                            // ...
                        };

How do you know which key was pressed in a keypress event?

event.key! For example: 

                        function onKeyPress(event) {
                            if (event.key === "l") {
                                // The L button was pressed!
                            }
                        };
How to create a new <p>-element in JS?

                    let newEl = document.createElement("p");

How to set the text inside it?

                    newEl.textContent = "Hello world!";

How to append it to an existing div?

                    divEl = ...;
                    divEl.appendChild(newEl);
How to add a class to an element?

                    newEl.classList.add('some-class');
                    newEl.classList.add('some-other-class');

How to remove a class?

                    newEl.classList.remove('some-other-class');
How to set the id of an element?

                    newEl.id = "my-new-element";
How to disable an element (e.g. button or input)?

                    newEl.setAttribue("disabled", true);

How to enable it again?

                    newEl.removeAttribute("disabled");
How to remove an element from the page?

                    newEl.remove();
