Comparisons ‚öñÔ∏è

  • Comparisons are what we put inside a if(...)
  • Are used to compare two values
  • They result in booleans!
    • 
                        let x = prompt("Please enter the first number");
                        let y = prompt("Please enter the second number");

  • We can also put the result into a variable:
    •

                        alert("z is a " + typeof(z) + " and contains " + z);

Types of comparisons

  • We've seen == (equals)
  • < (smaller than)
  • > (larger than)
  • <= (smaller or equal than)
  • >= (larger or equal than)
  • != (not equal)

Guess the results:

  • alert(12 > 33);
  • alert(12 < 33);
  • alert(12 < 12);
  • alert(12 <= 12);
  • alert(12 == 33);
  • alert(12 != 33);
  • alert("bla" == "bla");
  • alert("bla" != "bla");
  • alert("bla" == 12);
  • There are many other ways to create complex comparisons!
  • We will cover them later üòÅ
