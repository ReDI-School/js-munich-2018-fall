  • The weird JS jargon "alert" is a special kind of what?
  • It is a function
  • You will use functions ALL THE TIME
  • A reusable chunk of code that does something

Who and what defines a function?

  • Javascript framework itself 💻
  • You 👩🏻 👨🏻

  • What do you call the text inside the quotation marks ""?
  • It is a string
  • Always wrap a string in single or double quotes
let myText = "hello, world!";




  • Remember variables? 📦
  • How do we name a variable? There are a few rules.

Naming variables

let myText = "hello, world!";
  • Length:
  • 1 to few thousands
  • Restriction:
  • Can start with "_" or "$" but no number
  • What else?
  • That's it! Other than the first character, any combination of lower case or upper case letters, number, "_" or "$"

Possible variable name examples:

let myText;
let $;
let r8;
let _counter;
let thisIsALongVariableName_butItCanBeLonger;
let __$abc;
let oldSchoolName;

What makes a good variable name?

  • Is xxx a good variable name?

  • Is age a good variable name?
  • → Name should describe what is stored in the variable!
  • → Otherwise code will be hard to read and understand
  • In JavaScipt, it is common to use camelCase 🐫
  • "blog post title" becomes "blogPostTitle"
  • "number of users" becomes ...?

Let's say I want to find the distance a space ship travels

------------> 🚀

  • distance = speed * time
  • In JavaScript:
  • let speed = 10;
let time = 5;

And if I have more values?

let speed1 = 56;
let time1 = 7;
alert(speed1 * time1);

let speed2 = 12;
let time2 = 3;
alert(speed2 * time2);

let speed3 = 9;
let time3 = 88;

let speed4 = 64;
let time4 = 90;

Awful! 😫

So much repeated code!

Solve it by Function!

function showDistance (speed, time) {
  alert(speed * time);
}

showDistance(56, 7);
showDistance(12, 3);
showDistance(9, 88);
showDistance(64, 90);

Dive in:

  • Code a function named askName
  • Prompts the user to "Enter name"
  • Then assigns the answer to userName, which hasn't been declared beforehand
  • Then use alert to show the name.
  • Don't forget to make the function call.

function askName() {
  let userName = prompt("Enter name");
  alert(userName);
}
askName();
