Events

  • How to run a function when a button is clicked?

    
                                <button id="button">Beep</button>
                            
    
                                function onClick() {
                                    // ...
                                }
                                let buttonElement = document.getElementById("button");
                                buttonElement.addEventListener("click", onClick);

  • How to run a function when an input is edited?

    
                                <button id="button">Beep</button>
                            
    
                                function onInput() {
                                    // ...
                                }
                                let inputElement = document.getElementById("input");
                                inputElement.addEventListener("input", onInput);

                    buttonElement.removeEventListener("click", onClick1);

Event Object

  • Each event listener function gets an argument: The Event Object!
  • This event object has all the information about the event
  • Examples:
    • event.target has the exact target element of the event
    • event.key stores which keyboard key was pressed
