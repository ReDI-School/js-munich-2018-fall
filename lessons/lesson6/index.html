A quick recap:

Review

  • In Boolean logic, a statement can be either true or false
  • It rains today.
  • In javascript: 
    
                            if (x==4) {
                                do_something
                            }
  • Not surprising but we can do more

Logical operators

  • It rains today AND my feet are getting wet
  • It rains today OR my feet are getting wet
  • In javascript: 
    
                            if (x == 4 || y > 0) {
                                do_something;
                            }
  • Often we need to express more complex logic than
    x > y
  • We need to combine conditions together
  • AND: &&
    • Example: (x==4 && y==1)
  • OR: ||
    • Example: (x==4 || y==1)
  • NOT: !
    • Example: (x==4 && !(y==1))

&&

  • When the first expression AND the second expression both evaluate to true
  • Example: 
    
                            let x = 4; let y = 1;
                            alert(x==4 && y==1); 
                            alert(x==3 && y==1);
                            alert(x==1 && y==0);

||

  • When either the first expression OR the second expression both evaluate to true
  • Example: 
    
                            let x = 4; let y = 1;
                            alert(x==4 || y==1); 
                            alert(x==3 || y==1);
                            alert(x==1 || y==0);

!

  • When the containing expression is false then it is true. In other word it negates the result.
  • Example: 
    
                            let x = 4; let y = 1;
                            alert(x==4 && !(y==1)); 
                            alert(x==3 && !(y==1));
                            alert(x==1 && !(y==0));
                            alert(x==4 && !(y==0));

Guess the results:

  • alert(2 > 1 && 2 < 3);
  • alert(2 > 1 && 2 > 3);
  • alert(2 > 1 || 2 > 3);
  • alert(2 > 1 && !(2 > 1));

You can build it up

  • 
                            if (x==4 && (!(y==1) || z==0)) {   
                                do_something;
                            }
  • What is the result? 
    
                            let x = 3; let y = 9; let z = 9
                            alert(y >= x || (x===y && y > z));
                        
  • 
                            let x = 3; let y = 9; let z = 9
                            alert((y >=x && y < z) && !(x===y && y > z));
                        
  • 
                            let x = 3; let y = 9; let z = 9
                            alert((y === x || y === z) && !(x===y && y > z));

What is a FOR loop?

The for loop is the most often used kind of loop.

example


                    for (let i = 0; i < 3; i++) {
                        // shows 0, then 1, then 2
                        alert(i);
                    }

  • e.g. 2x1=2
    2x2=4
    2x3=6
    etc... until 2x10=20
  • Remember to use console.log("xxxxx")

                    function printMultiplicationTable(number) {
                        console.log("MultiplicationTable of: " + number)
                        for (let j = 1; j <= 10; j++) {
                            console.log(number + " x " + j + " = " + (number * j));
                        }
                    }
                    printMultiplicationTable(2);

Another kind of loop

The "while" loop

While the condition is true, the code from the loop body is executed.

the syntax is


                    while (condition) {
                        // code
                        // so-called "loop body"
                    }

let's try it!


                    let i = 0;
                    while (i < 3) { // shows 0, then 1, then 2
                        alert( i );
                        i++;
                    }
