Logical operators

  • Often logic is too complex for a simple condition
  • Need to combine conditions together
  • and - &&
  • or - ||
  • not - !

Guess the results:

  • alert(2 > 1 && 2 < 3);
  • alert(2 > 1 && 2 > 3);
  • alert(2 > 1 || 2 > 3);
  • alert(!(2 > 1));

another option is

The "while" loop

While the condition is true, the code from the loop body is executed.

the syntax is


	while (condition) {
		// code
		// so-called "loop body"
	}

let's try it!


	let i = 0;
	while (i < 3) { // shows 0, then 1, then 2
		alert( i );
		i++;
	}
