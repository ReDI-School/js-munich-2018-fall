Recap:

what is a function?

arguments are between the () separated by ","


	function showDistance(speed, time){
		alert(speed * time);
	}
	showDistance(56,7);

let's try to change something

now we use return inside the function


	function showDistance(speed, time){
		return (speed * time);
	}
	showDistance(56,7);

why?

we want to calculate the distance between multiple points


	let total_distance = 0;
	
	function showDistance(speed, time){
		return (speed * time);
	}

	total_distance = total_distance + showDistance(56,7);
	total_distance = total_distance + showDistance(42,8);
	total_distance = total_distance + showDistance(37,2);
	total_distance = total_distance + showDistance(67,5);

	alert("you run for " + total_distance + " metres");

As time goes on, the program becomes more and more complex. It becomes necessary to add comments which describe what happens and why.

  • One-line comments start with two forward slash characters //
    • 
	// This comment occupies a line of its own
	alert('Hello');

	alert('World'); // This comment follows the statement
  • Multiline comments start with a forward slash and an asterisk /* and end with an asterisk and a forward slash */
    • 
	/* An example with two messagxes.
	This is a multiline comment.
	*/
	alert('Hello');
	alert('World');
